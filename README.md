# Python Calculator

A simple, user-friendly command-line calculator built with Python that performs basic arithmetic operations.

## Features

- **Addition**: Add two numbers
- **Subtraction**: Subtract one number from another
- **Multiplication**: Multiply two numbers
- **Division**: Divide one number by another (with zero-division protection)
- **Error Handling**: Robust input validation and error handling
- **User-Friendly Interface**: Clear prompts and formatted output
- **Continuous Operation**: Perform multiple calculations without restarting

## Requirements

- Python 3.x

## Installation

1. Clone the repository:
```bash
git clone https://github.com/NusratBegum/Python-Calculator.git
```

2. Navigate to the project directory:
```bash
cd Python-Calculator
```

## Usage

Run the calculator by executing:
```bash
python index.py
```

Or:
```bash
python3 index.py
```

### How to Use

1. When the calculator starts, you'll see a menu with operation options
2. Select an operation by entering the corresponding number (1-5):
   - 1: Addition
   - 2: Subtraction
   - 3: Multiplication
   - 4: Division
   - 5: Exit
3. Enter the first number when prompted
4. Enter the second number when prompted
5. The result will be displayed
6. You can perform another calculation or exit the program

The calculator includes error handling for:

- Non-numeric input
